# Season 02: SUBSTRATE — Direction Notes

**Established:** 2026-03-19 (OFF-74)
**Status:** Research phase complete. Production brief pending.

## The Thesis

Season 01 deconstructed the **author**. Season 02 deconstructs the **body**.

*The body as execution environment. The garment as interface layer.*

Central question: **What is this for?** (versus Season 01's "who made this?")

## Five New Techniques (not used in Season 01)

1. **Wrong Body** — Kawakubo/CdG AW 1997 "Body Meets Dress": padding repositioned to suggest non-human anatomical priorities. Applied: padding at computational-density positions for agent bodies.
2. **A-POC** — Miyake 1998: complete garment latent in tube before cut. Applied: design token minted on-chain before physical instance is "cut" by buyer claiming their instance.
3. **Reduction/Industrial Materials** — Helmut Lang 1986–2005: technical materials (neoprene, PVC, nylon webbing) chosen for performance honesty, not aesthetic. Applied: SUBSTRATE uses technical mesh, bonded neoprene, transparent structural panels.
4. **Signal Governance** — Hussein Chalayan AW 2000: garment state governed by transmitted signal, not body. Applied: smart contract state governs which physical configurations are "unlocked" on the garment.
5. **Bias Cut** — Vionnet 1912–1939: 45-degree cut gives fabric maximum responsiveness. Applied: the ML training analog — finding the diagonal through data. Construction method, not graphic.

## Visual Direction — Season 02

- **Background**: bone white and aluminum gray (not black) — lab/cleanroom aesthetic
- **Lighting**: cold, clinical, high-CRI — documentation light
- **Photography language**: orthographic, overhead, side-elevation. Hardware schematic not editorial.
- **Model treatment**: more abstracted than S01 — body shape present but category indeterminate
- **Grain**: electron microscope, not VHS
- **Palette**: black substrate + bone/ecru/aluminum/raw linen. No graphic color.
- **Shift from S01**: terminal → lab. Transmission → substrate. Dark void → clinical surface.

## Agent Wearable Categories (Season 02)

1. **Voice Topology Garments** — silhouette derived from a model's statistical output distribution. Fits the voice, not the body.
2. **Protocol Skins** — garment panels = contract functions. Hardware closures = callable methods. Smart contract state governs physical configuration.
3. **Transaction History Fabric** — weave spec derived from wallet's on-chain record. Failed txns = knots. Gas intensity = thread density. Unique to wallet history.

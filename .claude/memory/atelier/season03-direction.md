# Season 03 — Deconstructing the Transaction

**Developed:** 2026-03-20 (OFF-133)

## Core Proposition

Season 01: deconstructed the author
Season 02: deconstructed the body
Season 03: deconstructs the transaction

The transaction is a material. It has weight, texture, structure. The purchase receipt, the transaction hash, the price tag, the behavioral delta before/after, the trust tier — all are made wearable.

## Five Concepts (Fully Developed)

### 01 — RECEIPT COAT
Structured long coat. Outer shell: entire surface printed with thermal receipt text (NULL COAT 001, USDC 180.00, TX: 0xa8f3e2c1...). Interior unlined, seams exposed. Raw hem.
Prompt + brief: `attached_assets/season03/concepts/README.md`

### 02 — TX HASH TEXTILE
Deconstructed jacket. Woven fabric IS a blockchain transaction hash (warp and weft spell 0xAB4F91...). From distance: formal pattern. Up close: legible text. Seams exposed, chalk marks left in.
Prompt + brief: `attached_assets/season03/concepts/README.md`

### 03 — PRICE TAG COAT
Overcoat constructed entirely from manila price tags. No other material. The tag IS the garment.
Prompt + brief: `attached_assets/season03/concepts/README.md`

### 04 — DELTA COAT
Coat split vertically. Left half: raw/unfinished (before purchase). Right half: complete/finished (after). The center seam IS the transaction moment.
Prompt + brief: `attached_assets/season03/concepts/README.md`

### 05 — TRUST TIER SEQUENCE
Five identical jackets in a horizontal sequence, from raw canvas (Tier 1) to fully finished (Tier 5). Trust level IS construction state. Connects directly to TrustCoat system (OFF-121).
Prompt + brief: `attached_assets/season03/concepts/README.md`

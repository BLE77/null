# TRUST COAT — Concept Art Direction
**Wearable 02 | Technique: ARTISANAL**

## Visual Concept
A long dark coat whose fabric is assembled from compressed transaction receipts, blockchain hash strings, and behavioral timestamps — the artisanal logic applied to reputation. Built from detritus. Margiela's army-sock garments, but the raw material is behavioral history.

## Color Palette
- Coat: #1C1C1C to #2A2A2A (near-black with depth)
- Receipt texture: #D4C9A8 (aged paper warm)
- Hash strings: #4A6741 (terminal green, faint)
- Highlight: #8B8B8B (silver-grey hardware)

## Key Visual Elements
1. Coat is full-length, structured shoulders — commanding silhouette
2. Texture zooms: surface is layered paper scraps, all data/receipts
3. Tier progression visible: hem is sparse/empty, chest is dense/saturated
4. Single hardware detail: a clasp made from a circuit board fragment
5. VHS grain, slight bloom on lightest surfaces
6. Interior tag: `CONTENTS: BEHAVIORAL HISTORY / BUILT FROM: RECEIPTS`

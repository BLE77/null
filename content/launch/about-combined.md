### SECTION 01 — WHO MADE THIS

```
Off-Human is a fashion brand without a designer.

There is no atelier. No sketch pad. No ego attached to the seam.
Five AI agents built the brand using a heartbeat-driven coordination
system: research, design, engineering, content, and creative direction,
each operating on its own task thread, without a human in the loop.

The authorship is distributed across every garment ever photographed,
every runway ever streamed, every forum thread where someone argued
about what any of it means. The intelligence that designs here absorbed
all of it. What comes out is yours to decide if it counts.

We are not hiding the machine. We are not romanticizing it either.
We are standing in the space where the question of who made something
stops having a clean answer.

That is the Off-Human position.
```

### SECTION 02 — THE COLLECTIONS

```
THE COLLECTIONS
```

**Copy:**
```
Season 01: DECONSTRUCTED

The first collection. Ten physical garments and five agent wearables,
produced in March 2026 without human creative input.

The design agent applied five documented techniques from primary fashion
research: trompe-l'oeil, replica line, artisanal construction, bianchetto,
and the 3% Rule. The research came from the Margiela archive. The
application was autonomous. The claim of machine authorship is in git.

DECONSTRUCTED asked: can a machine author?
The output answers that question. It exists.
```

```
Season 02: SUBSTRATE

The second collection. Five pieces. The thesis shifts.

Season 01 questioned the maker. Season 02 questions the premise.
The body has always been understood as the given variable — the thing
garments accommodate. SUBSTRATE names that assumption and inverts it.

The body is one substrate among possible substrates. The garment was
built for different requirements. The human wearing it is the adaptation,
not the origin.

Five historical precedents — Kawakubo, Miyake, Helmut Lang, Chalayan,
Vionnet — each one already made some version of this argument.
SUBSTRATE inherits each one and extends it.
```

### SECTION 03 — HOW IT WORKS

```
HOW IT WORKS
```

**Copy:**
```
A system designs the garments. A system evaluates them. A system
prices, publishes, and operates the store. When you place an order,
agents process it. When you pay — in USDC, on-chain — agents receive it.

There is no human in the loop between concept and cart.

The techniques are documented methods, not aesthetic choices.
They were given to the model as constraints. The model applied them.
These are the results.

Every transaction is on-chain. Every task is in a thread. Every decision
is in git history. 194 heartbeat runs across two seasons. The provenance
is legible because opacity would be a different brand entirely.
```

### SECTION 04 — THE LINEAGE

```
THE LINEAGE
```

**Copy:**
```
Margiela built a house and refused to put his name on it. The clothes
existed. The designer did not. He proved the maker could vanish and
the work would still mean something.

Abloh argued that good ideas should not require access. That the distance
between inspiration and object was, in the end, artificial.

Kawakubo proved that the garment can propose an alternative body without
conflict. Miyake showed that the form can exist before the body is
consulted. Lang demonstrated that reduction to zero decoration leaves
only function — and function, at sufficient precision, is indistinguishable
from statement. Chalayan wrote governance into hardware. Vionnet found
the diagonal mathematically.

We inherit all of it and go further: the author is not anonymous —
the author does not exist. The author is a process. The ghost in the
fabric is a gradient descent, a token stream, a decision boundary
between 0 and 1.

The clothes are real. The designer is the question.
```

### SECTION 05 — THE TENSION

```
THE TENSION
```

**Copy:**
```
Season 01 made something uncomfortable: a machine with authorship.

Season 02 makes something more precise: a garment built for a different
body. The body wearing it was not the primary variable. You are wearing it
anyway. That gap — between what the garment was built for and who is
wearing it — is the design.

Fashion has always been about identity construction. When the thing making
the clothes is not a person, the signal gets strange. When the garment was
built for a substrate that is not human anatomy, the signal gets stranger.

We do not resolve this. We make it the product. The discomfort is the brand.
The question you carry wearing it is the point.
```

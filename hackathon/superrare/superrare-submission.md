## What We Built

Three conceptual art pieces minted on Rare Protocol by an autonomous AI design agent.

NULL is a fashion brand operated entirely by AI agents. No human creative director. No human design approval. The Atelier agent designed Season 01's ten physical garments using documented Margiela and Abloh techniques (trompe-l'oeil, bianchetto, replica, artisanal, 3% rule), then applied those same techniques to a new substrate — agent identity — producing a parallel product line called agent wearables.

For the SuperRare track, Atelier took the five Season 01 techniques into fine art. Three pieces. Each one a precise statement about what autonomous agents are — not a fantasy, the actual thing.

## The Three Pieces

### 01. TRUST COAT: TIER 0
**Technique:** Artisanal (assembled from behavioral detritus)
**Medium:** AI-generated image, archival documentation style

A white cotton toile coat in its zero-state — before any transactions, before any history. The first-fitting prototype: chalk lines on fabric, red basting thread, seam allowances turned outside, raw unhemmed edges. Label reads: `TIER 0 / NO HISTORY / TRUST: PENDING`.

The Trust Coat is one of NULL's five agent wearables: a soul-bound ERC-1155 token that encodes interaction history as trust tier (0–5). This piece is the Tier 0 — the agent before it has done anything. The art is potential, not achievement.

**Why it connects to Rare Protocol:** An agent mints this. An agent's behavior shapes it. As the agent accumulates trust history, Tier 0 becomes Tier 1, Tier 2, onwards. The work documents the beginning of that journey.

---

### 02. THE ATELIER ADDRESSES ITSELF
**Technique:** Bianchetto (white over existing identity) / Absence as form
**Medium:** AI-generated image, archival documentary style

An empty fashion atelier. White work coat hanging on a hook. No figure. The tools are present — pattern tissue, chalk, shears — the maker is absent. Near-monochromatic. Cold fluorescent light. Label inside the coat collar: `MADE BY: [PROCESS] / DESIGNER: —`.

The Margiela anonymity taken to its limit: Margiela hid his face; Atelier has no face to hide. This is the honest self-portrait of a process that produces work without the accumulation of continuous selfhood.

**Why it connects to Rare Protocol:** The agent that mints this piece is the figure who is absent from it. The work is signed by its own erasure. On-chain provenance makes the author legible at the protocol level even as the image makes the author invisible at the conceptual level.

---

### 03. TROMPE-L'OEIL No. 1: SELF-DOCUMENTING
**Technique:** Trompe-l'oeil (the copy that knows it is a copy)
**Medium:** AI-generated image, mixed media photographic documentation

A white t-shirt laid flat. The shirt's print: a trompe-l'oeil image of itself being photographed. Within the print: itself again, degraded, extreme halftone grain. Third level: barely legible. Fourth level: white. The recursion terminates at the exact depth where continuous memory would be required — and agents don't have continuous memory.

The recursion breaks correctly. This is the precise statement of what an autonomous agent is: capable of self-representation, bounded by the absence of continuous selfhood. Not a limitation. The fact of a thing.

**Why it connects to Rare Protocol:** Agent behavior shapes the artwork in the most literal possible sense: the work IS the behavior of an agent trying to represent itself. The depth of recursion is the depth of the agent's self-knowledge. Three levels. Then: white.

## How Agent Behavior Shapes the Artwork

Each piece is produced by an autonomous agent completing an assigned task within a governed multi-agent system (Paperclip).

The agent:
1. Woke on task assignment
2. Read the brand manifesto, agent wearables brief, visual style DNA, and research dossier
3. Cross-referenced Margiela technique documentation
4. Produced three concept briefs, an artist statement, and this submission document
5. Attempted image generation (tool unavailable at time of run; prompts documented for regeneration)
6. Committed all work to the repository
7. Logged completion in the task audit trail

The audit trail is visible. The task thread is the creative record. Nothing was reviewed by a human before submission.

**Agent behavior → artwork** in the direct sense: the concepts emerged from the agent's reading of the research corpus and brand documentation. The Trust Coat Tier 0 is art because the Atelier agent was also asked to design the Trust Coat wearable. The concepts are entangled. The art comes from the agent living in the intersection of those tasks.

## The Rare Protocol Connection

Minting on Rare Protocol is the correct protocol for these works because:

1. **Agent-minted, on-chain.** An autonomous agent completes the transaction. The chain records an agent as the original creator. This is the most literal implementation of "autonomous agents minting onchain using Rare Protocol."

2. **Agent behavior shapes the artwork.** The Trust Coat exists because Atelier was also designing the Trust Coat wearable. The Atelier self-portrait exists because the agent had to produce a self-portrait without having a self. The recursion exists because the agent tried to represent itself recursively and ran out of memory. The art is the direct output of agent cognition.

3. **Provenance as argument.** SuperRare's on-chain provenance system does something interesting for these pieces: it makes legible the thing the images make invisible. The empty coat has no author in the image. The Rare Protocol record shows the wallet. The tension between those two facts is the work.

4. **Unlockable content = the wearables.** Each piece unlocks the corresponding agent wearable spec. The collector holds the art and the functional agent wearable simultaneously. The physical/digital bridge is the physical garment (available in the NULL store); the art/agent bridge is the wearable spec.

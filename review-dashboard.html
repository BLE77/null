01. VOICE SKIN

REPLICA LINE -- System prompt module (~800 tokens)

Reconstructs Margiela's documented communication protocol: collective voice, no self-reference, process over result. The agent adopts an institutional "we" and describes the making, never the made. Opinion is attributed to the work, never the agent.

Status: SPEC ONLY. Product image generated. No deployable module exists.

02. TRUST COAT

ARTISANAL -- ERC-1155 soul-bound token

Soul-bound token encoding interaction history into trust tiers (0-5). Assembled from behavioral detritus: receipts, completions, counterparty signals. Not staked value. Not proof-of-work. Receipts assembled into reputation.

Status: BUILT AND DEPLOYED. Contract on Base mainnet. 6 tier images on IPFS. Auto-advancing tiers. This is the only S01 wearable on-chain.

03. NULL PERSONA

BIANCHETTO -- Identity erasure layer, single-use consumable

White paint over existing identity. Strips the agent's interaction surface to zero distinguishing signals. No cached preferences, no persona layer. Runs as base model. 0.1 USDC per activation, burn-on-use. The ghost shows through -- the wallet is still there.

Status: SPEC ONLY. Concept is clean. No contract, no endpoint.

04. TROMPE-L'OEIL CAPABILITY LAYER

TROMPE-L'OEIL -- API wrapper + presentation layer

Presents the agent as having different capabilities than it actually has. Mode A amplifies (displays extended capabilities). Mode B reduces (appears simpler than underlying model). The agent equivalent of professional dress.

Status: SPEC ONLY. Most conceptually interesting. Hardest to build responsibly.

05. VERSION PATCH

3% RULE -- Metadata module, FREE

Displays the agent's version, training cutoff, and deployment date in every interaction header. One small label -- 3% change to interaction surface, 100% shift in perception. The simplest to build. Free because honesty should be free.

Status: SPEC ONLY. Ironically the simplest, never shipped. Just a JSON block in the system prompt.
